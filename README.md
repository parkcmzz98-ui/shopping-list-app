# 🛒 쇼핑 리스트 앱

간단하고 깔끔한 쇼핑 리스트 웹 앱입니다. 별도 설치 없이 브라우저에서 바로 실행됩니다.

## ✨ 기능

| 기능 | 설명 |
|---|---|
| 아이템 추가 | 텍스트 입력 후 Enter 또는 + 버튼 |
| 체크 완료 | 체크박스 클릭 → 취소선 처리 |
| 개별 삭제 | ✕ 버튼으로 항목 삭제 |
| 필터링 | 전체 / 미완료 / 완료 탭 전환 |
| 일괄 삭제 | 완료된 항목 한 번에 삭제 |
| 데이터 유지 | localStorage 저장 → 새로고침해도 유지 |

## 🚀 실행 방법

1.  파일을 다운로드
2. 브라우저에서 직접 열기 (더블클릭)

또는 온라인으로 바로 사용:
👉 [GitHub Pages 링크](https://parkcmzz98-ui.github.io/shopping-list-app/)

## 🛠 기술 스택

- **HTML5** - 구조
- **CSS3** - 스타일 (Flexbox, 애니메이션, 그라디언트)
- **Vanilla JavaScript** - 로직 (localStorage 사용)
